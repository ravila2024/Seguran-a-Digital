# 🔒 Gerador de Senhas Seguras

Projeto desenvolvido para o curso da **Alura + SEED/PR**, com o tema:
**Segurança digital: utilizando matemática para programar senhas seguras**

---

## 📌 Sobre o projeto

Este projeto é um gerador de senhas seguras que utiliza conceitos de **matemática e programação** para criar senhas fortes e difíceis de serem descobertas.

Ele demonstra como a matemática está presente na segurança digital por meio de:
- Combinações possíveis
- Probabilidade
- Crescimento exponencial de possibilidades

---

## 🧠 Conceitos utilizados

- Matemática aplicada à segurança digital
- Potenciação (base^expoente)
- Números aleatórios
- Lógica de programação
- Segurança da informação

---

## ⚙️ Funcionalidades

- Gerar senhas aleatórias
- Escolher tamanho da senha (8 a 20 caracteres)
- Incluir:
  - Letras maiúsculas
  - Letras minúsculas
  - Números
  - Símbolos
- Avaliação da força da senha
- Cálculo de combinações possíveis
- Botão para copiar senha

---

## 📊 Parte matemática

A segurança da senha é baseada no número de combinações possíveis:

> combinações = (quantidade de caracteres possíveis) ^ (tamanho da senha)

Exemplo:
Se temos 70 caracteres possíveis e uma senha de 12 posições:

70¹² ≈ bilhões de combinações possíveis

Isso torna ataques por tentativa e erro extremamente difíceis.

---

## 🚀 Como usar

1. Baixe os arquivos do projeto
2. Abra o arquivo `index.html` no navegador
3. Clique em "Gerar Senha"
4. Copie sua senha segura

---

## 🌐 Publicação no GitHub Pages

1. Crie um repositório no GitHub
2. Envie os arquivos:
   - index.html
   - style.css
   - script.js
   - README.md
3. Vá em **Settings > Pages**
4. Em "Source", selecione **main / root**
5. Copie o link gerado e envie na atividade
